Cascading Style Sheets Book

Cascading Style Sheets (CSS) is a style sheet language used for
describing the look and formatting of a document written in a
markup language. Although most often used to change the style of
web pages and user interfaces written in HTML and XHTML, the
language can be applied to any kind of XML document, including
plain XML, SVG and XUL. Along with HTML and JavaScript, CSS
is a cornerstone technology used by most website to create visually
engaging webpages, user interfaces for web application, and user
interfaces for many mobile application.[1]
